A sophisticated, modern web application designed for high-end hotel management and guest experiences. Built with a focus on premium aesthetics, seamless performance, and responsive design.

## ✨ Key Features

- **💎 Premium UI/UX**: Clean, elegant design using Radix UI and Tailwind CSS.
- **🚀 Ultra-Fast Performance**: Built on Vite for lightning-fast development and optimized production builds.
- **📱 Fully Responsive**: Optimized for desktop, tablet, and mobile devices.
- **🔐 Secure Authentication**: Integrated with Supabase for robust user management.
- **🖼️ Dynamic Hero Section**: Beautiful animations powered by Framer Motion.
- **📊 Real-time Data**: Syncs with Supabase for up-to-the-minute management.

## 🛠️ Tech Stack

- **Frontend**: React 18, TypeScript, Vite
- **Styling**: Tailwind CSS, Shadcn/UI, Lucide React
- **Animations**: Framer Motion, Swiper
- **Backend/Auth**: Supabase
- **Deployment**: Vercel

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or bun

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/tanmaypal10/GRANDHOTLE.git
    cd GRANDHOTLE
    ```

2.  **Install dependencies**:
    ```bash
    cd FRONTEND
    npm install
    ```

3.  **Set up Environment Variables**:
    Create a `.env` file in the `FRONTEND` directory and add your Supabase credentials:
    ```env
    VITE_SUPABASE_URL=your_supabase_url
    VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
    ```

4.  **Run the development server**:
    ```bash
    npm run dev
    ```

## 📦 Deployment

The project is configured for seamless deployment on **Vercel**. 

- **Frontend Root**: `FRONTEND`
- **Build Command**: `npm run build`
- **Output Directory**: `dist`
